# 02_BASE_POO -- Programação Orientada a Objetos

## 📌 Descrição

Conjunto de exercícios focados nos fundamentos de Programação Orientada
a Objetos em Java.

------------------------------------------------------------------------

## 📚 Exercícios

### 🔹 Exercício 35 -- Calculadora

Programa que realiza: - Soma - Subtração - Multiplicação - Divisão -
Potência - Raiz quadrada

Com validação de entradas e tratamento de erros.

------------------------------------------------------------------------

### 🔹 Exercício 36 -- Conversor de Moedas

Conversão entre: - Real (BRL) - Dólar (USD) - Euro (EUR)

Utiliza `DecimalFormat` para formatação monetária e tratamento de
exceções.

------------------------------------------------------------------------

### 🔹 Exercício 37 -- Sistema Bancário

Funcionalidades: - Saque (com validação de saldo) - Depósito - Consulta
de saldo - Empréstimo (+12% de acréscimo)

Inclui controle de regras de negócio e tratamento de erros.

------------------------------------------------------------------------

### 🔹 Exercício 38 -- SmartHome 2.0 (Interfaces)

Implementação de Interface `DispositivoEletronico` com: -
LampadaInteligente - SmartTV - CaixaDeSom

Demonstração de polimorfismo via classe de teste.

------------------------------------------------------------------------

### 🔹 Exercício 39 -- Java Quest (Classe Abstrata)

Criação da classe abstrata `Personagem` e subclasses: - Guerreiro -
Mago - ProgramadorJava

Exploração de herança e métodos abstratos.

------------------------------------------------------------------------

### 🔹 Exercício 40 -- Sistema de Gestão de Frotas

Expansão de sistema legado com novas classes: - Carro - Moto

Aplicação de herança, polimorfismo e integração com estrutura existente.

------------------------------------------------------------------------

## 🧠 Conceitos Aplicados

-   Encapsulamento
-   Herança
-   Polimorfismo
-   Abstração
-   Interfaces
-   Tratamento de Exceções
-   Regras de Negócio

------------------------------------------------------------------------

## 🛠 Tecnologias

-   Java 21
-   NetBeans
-   API padrão Java

------------------------------------------------------------------------

Este módulo representa a consolidação dos fundamentos de POO aplicados
em cenários práticos.
